# Sistema de Adopción de Mascotas

Aplicación web para la gestión de mascotas, clientes, adopciones, visitas y documentos.

## Tecnologías
- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage (persistencia)

## Estructura

/project
index.html
/js
ENV.js
librerias.js
librerias_adopcion.js
persistencia.js
integracion.js


## Funcionalidades
- CRUD de mascotas, clientes y adopciones
- Gestión de visitas
- Subida de documentos (simulada)
- Dashboard con estadísticas

## Ejecución
Abrir `index.html` en el navegador.

## Notas
- No requiere backend
- Persistencia local en el navegador
